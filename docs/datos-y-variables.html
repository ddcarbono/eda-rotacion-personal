Análisis Exploratorio de Rotación de Personal

Capítulo 2 Datos y variables

2.1 Fuentes de datos

Los datos provienen de dos extracciones del sistema de información de nómina y recursos humanos de la empresa:

  • Empleados activos: snapshot del personal con vínculo activo a marzo de 2026.
  • Retiros por renuncia voluntaria: registro de empleados que presentaron renuncia voluntaria entre marzo de 2025 y marzo de 2026.

Ambas fuentes comparten la misma estructura de 21 variables, lo que permite su integración directa en un único dataset.

2.3 Procesamiento y decisiones

A partir del dataset original se realizaron las siguientes transformaciones antes del análisis:

Exclusión de aprendices. Se eliminaron los registros con función == "APRENDIZ" (38 registros). Los aprendices tienen un vínculo de formación práctica con duración definida, por lo que su desvinculación al finalizar el período no constituye renuncia voluntaria en sentido estricto.

Tratamiento de valores nulos. Las columnas fecha_retiro y tipo_retiro contenían el carácter "-" en los registros de empleados activos. Estos valores fueron reemplazados por NA.

Recodificación de estado civil. Las categorías Divorciado y Viudo presentaban frecuencias de 1 caso cada una. Se agruparon junto con Separado en la categoría Otro.

Construcción de nivel de función. A partir de la variable función se construyó nivel_funcion con cinco niveles jerárquicos: Operativo, Auxiliar, Asistente, Coordinador y Jefe/Dirección.

df <- df_raw %>%
  filter(función != "APRENDIZ") %>%
  mutate(
    estado        = factor(estado, levels = c("Activo", "Retirado")),
    género        = factor(género, levels = c("Masculino", "Femenino")),
    tipo_personal = fct_collapse(tipo_personal,
      "Administrativos" = c("Administrativos", "Administ/Operativ.",
                            "Coorporativo", "Corporativo"),
      "Operativos"      = "Operativos"),
    contrato      = factor(contrato),
    fecha_retiro  = if_else(fecha_retiro == "-", NA_character_, fecha_retiro),
    tipo_retiro   = if_else(tipo_retiro  == "-", NA_character_, tipo_retiro),
    antig_m       = as.numeric(antig_m),
    edad          = as.numeric(edad),
    est_civil     = case_when(
      estado_civil == "Soltero"     ~ "Soltero",
      estado_civil == "Casado"      ~ "Casado",
      estado_civil == "Union Libre" ~ "Unión Libre",
      TRUE                          ~ "Otro"
    ) %>% factor(levels = c("Soltero", "Casado", "Unión Libre", "Otro")),
    nivel_funcion = case_when(
      función == "OPERATIVO"                                            ~ "Operativo",
      función == "AUXILIAR"                                             ~ "Auxiliar",
      función == "ASISTENTE"                                            ~ "Asistente",
      función == "COORDINADOR"                                          ~ "Coordinador",
      función %in% c("JEFE", "DIRECTOR", "GERENCIA", "VICEPRESIDENTE") ~ "Jefe/Dirección"
    ) %>% factor(levels = c("Operativo", "Auxiliar", "Asistente",
                             "Coordinador", "Jefe/Dirección"))
  )

2.7 Análisis de valores nulos
