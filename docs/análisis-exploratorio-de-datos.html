3.1 Tasas de rotación

3.1.1 Tasa general

La tasa de rotación voluntaria anual se calcula como el cociente entre el número de renuncias voluntarias en el período y el total de empleados activos al cierre del mismo.

tibble(
  Indicador = c(

3.1.2 Retiros por mes

El análisis mensual permite identificar si existe estacionalidad o períodos de mayor concentración de renuncias a lo largo del año.

df %>%
  filter(estado == "Retirado") %>%
  count(mes_retiro) %>%
  ggplot(aes(x = mes_retiro, y = n)) +
  geom_col(fill = "#405731", color = "#b08a2d", width = 25) +
  geom_text(aes(label = n), vjust = -0.5, size = 3.5, color = "#333333") +
  scale_x_date(date_breaks = "1 month", date_labels = "%b\n%Y") +
  scale_y_continuous(expand = expansion(mult = c(0, 0.15))) +
  labs(title    = "Renuncias voluntarias por mes",
       subtitle = paste0("Total del período: ", n_retirados, " renuncias"),
       x        = NULL,
       y        = "Número de renuncias") +
  tema

El análisis mensual revela una distribución irregular de las renuncias a lo largo del período. Enero de 2026 concentra el mayor número de retiros con 16 casos, seguido de marzo, julio y agosto de 2025 con 10 casos cada uno. Los meses de abril y mayo de 2025 registran la menor actividad con 4 renuncias cada uno, y septiembre de 2025 no presenta ningún retiro. No se observa un patrón estacional claro, aunque el pico de enero podría estar asociado a decisiones tomadas al cierre del año anterior.

3.1.3 Tasa de rotación por segmento

La tasa de rotación por segmento se calcula como el número de retirados en cada categoría dividido entre el total de empleados en esa categoría, expresado como porcentaje.

calcular_tasa <- function(var) {
  df %>%
    count(estado, {{ var }}) %>%
    pivot_wider(names_from = estado, values_from = n, values_fill = 0) %>%
    mutate(
      Total = Activo + Retirado,
      Tasa  = percent(Retirado / Total, accuracy = 0.1)
    ) %>%
    rename(Categoría = {{ var }})
}

bind_rows(
  calcular_tasa(tipo_personal) %>% mutate(Variable = "Tipo de personal"),
  calcular_tasa(género)        %>% mutate(Variable = "Género"),
  calcular_tasa(contrato)      %>% mutate(Variable = "Tipo de contrato"),
  calcular_tasa(est_civil)     %>% mutate(Variable = "Estado civil"),
  calcular_tasa(nivel_funcion) %>% mutate(Variable = "Nivel de función")
) %>%
  select(Variable, Categoría, Activo, Retirado, Total, Tasa) %>%
  kable(caption = "Tasa de rotación voluntaria por segmento") %>%
  kable_styling(bootstrap_options = c("striped", "hover", "condensed"),
                full_width = FALSE) %>%
  row_spec(0, background = "#405731", color = "white", bold = TRUE)

La categoría Otro registra una tasa de 0.0%, aunque este resultado debe interpretarse con cautela dado que agrupa categorías con muy pocos casos. Los activos presentan una distribución más amplia y uniforme, con mayor presencia en rangos de edad intermedios y altos. Esto sugiere que la renuncia voluntaria es un fenómeno predominantemente asociado a empleados jóvenes en etapas tempranas de su trayectoria laboral en la empresa.

El diagrama de dispersión muestra una correlación positiva entre edad y antigüedad en ambos grupos — a mayor edad, mayor antigüedad esperada. Sin embargo, los retirados se concentran en la esquina inferior izquierda del plano (jóvenes con poca antigüedad), mientras que los activos se distribuyen a lo largo de toda la diagonal. La pendiente de la línea de tendencia es similar entre grupos, pero los retirados muestran mayor dispersión relativa, lo que sugiere que algunos empleados jóvenes con baja antigüedad renuncian independientemente de su edad exacta.
