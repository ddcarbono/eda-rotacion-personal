Análisis Exploratorio de Rotación de Personal

Capítulo 4 Análisis Inferencial

El análisis inferencial busca determinar si las diferencias observadas en el capítulo anterior entre empleados activos y retirados son estadísticamente significativas o pueden atribuirse a fluctuación aleatoria. Se sigue la misma lógica de los análisis de referencia: primero se verifica el supuesto de normalidad para las variables numéricas, y según ese resultado se elige entre pruebas paramétricas o no paramétricas. Para las variables categóricas se aplica la prueba de chi-cuadrado o la prueba exacta de Fisher según corresponda.

4.1 Pruebas de normalidad

Antes de comparar edad y antigüedad entre grupos se evalúa si estas variables siguen una distribución normal dentro de cada grupo, mediante la prueba de Anderson-Darling.

vars_num <- c("edad", "antig_a")
grupos   <- c("Activo", "Retirado")

resultados_norm <- map_dfr(vars_num, function(var) {
  map_dfr(grupos, function(g) {
    x <- df %>% filter(estado == g) %>% pull(!!sym(var)) %>% na.omit()
    test <- ad.test(x)
    tibble(
      Variable  = var,
      Grupo     = g,
      N         = length(x),
      Estadístico = round(test$statistic, 3),
      `P-valor` = ifelse(test$p.value < 0.001, "< 0.001",
                         as.character(round(test$p.value, 3)))
    )
  })
})

resultados_norm %>%
  kable(caption = "Pruebas de normalidad Anderson-Darling por variable y grupo") %>%
  kable_styling(bootstrap_options = c("striped", "hover", "condensed"),
                full_width = FALSE)
Table 4.1: Pruebas de normalidad Anderson-Darling por variable y grupo
Variable Grupo N Estadístico P-valor
edad Activo 1251 13.856 < 0.001
edad Retirado 91 1.319 0.002
antig_a Activo 1251 65.516 < 0.001
antig_a Retirado 91 10.291 < 0.001

Dado que las pruebas de Anderson-Darling rechazan la hipótesis de normalidad en ambas variables y grupos (p < 0.05), se utilizan pruebas no paramétricas para las comparaciones entre grupos.

4.2 Comparación de edad entre grupos

Hipótesis:

  • \(H_0\): La distribución de la edad es igual en empleados activos y retirados.
  • \(H_1\): La distribución de la edad difiere entre empleados activos y retirados.
test_edad <- wilcox.test(edad ~ estado, data = df)

tibble(
  Variable    = "Edad",
  Prueba      = "Wilcoxon-Mann-Whitney",
  Estadístico = round(test_edad$statistic, 1),
  `P-valor`   = ifelse(test_edad$p.value < 0.001, "< 0.001",
                       as.character(round(test_edad$p.value, 3)))
) %>%
  kable(caption = "Prueba de Wilcoxon-Mann-Whitney para edad") %>%
  kable_styling(bootstrap_options = c("striped", "hover"),
                full_width = FALSE)
Table 4.2: Prueba de Wilcoxon-Mann-Whitney para edad
Variable Prueba Estadístico P-valor
Edad Wilcoxon-Mann-Whitney 81438.5 < 0.001

4.4 Asociación entre grupo y tipo de personal

Hipótesis:

  • \(H_0\): El tipo de personal es independiente del grupo (activo vs. retirado).
  • \(H_1\): Existe asociación entre el tipo de personal y el grupo.
tabla_tp <- table(df$estado, df$tipo_personal)
test_tp  <- chisq.test(tabla_tp)

tibble(
  Variable    = "Tipo de personal",
  Prueba      = "Chi-cuadrado",
  Estadístico = round(test_tp$statistic, 3),
  gl          = test_tp$parameter,
  `P-valor`   = ifelse(test_tp$p.value < 0.001, "< 0.001",
                       as.character(round(test_tp$p.value, 3)))
) %>%
  kable(caption = "Prueba chi-cuadrado: grupo vs. tipo de personal") %>%
  kable_styling(bootstrap_options = c("striped", "hover"),
                full_width = FALSE)
Table 4.4: Prueba chi-cuadrado: grupo vs. tipo de personal
Variable Prueba Estadístico gl P-valor
Tipo de personal Chi-cuadrado 0.201 3 0.977
